# Flight Price Data Preprocessing

## Project Overview
This project focuses on cleaning and preprocessing a flight price dataset to prepare it for machine learning model training.

The dataset contains information about airline, source, destination, duration, total stops and ticket price.

## Objectives
- Clean raw dataset
- Handle missing values
- Perform feature engineering
- Convert categorical variables into numerical format
- Prepare dataset for machine learning models

## Technologies Used
Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Scikit-learn  

## Data Preprocessing Steps

1. Data Cleaning
- Removed unnecessary columns
- Checked for missing values
- Standardized data formats

2. Feature Engineering
- Extracted hour and minute from few columns
- Converted categorical variables into numerical format using encoding

3. Data Transformation
- Prepared final dataset suitable for training machine learning models

## Project Outcome
The dataset was successfully cleaned and transformed into a structured format ready for machine learning algorithms.
